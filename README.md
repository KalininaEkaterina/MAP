Приложение «Расписание авиарейсов» разработано для Android и предназначено для хранения и просмотра данных о рейсах в базе SQLite.
В системе используются три основные сущности: аэропорты, типы самолетов и рейсы.

Приложение выполняет три ключевых запроса:

- вывод списка рейсов прибытия в выбранный аэропорт на заданную дату
- вывод информации о типах самолетов, связанных с выбранным аэропортом
- вывод рейсов по заданным пунктам вылета и прибытия на выбранную дату

Результаты отображаются в виде списков с несколькими колонками. Для списка рейсов реализован кастомный адаптер с графическим элементом: тип самолета автоматически отображается соответствующей иконкой.

Дополнительно реализованы отложенные напоминания через WorkManager (уведомления о рейсах), которые срабатывают даже при закрытом приложении.
